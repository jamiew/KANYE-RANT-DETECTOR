Kanye Rant Detector

F.A.T. Lab, 2009

@kanyerants

A Twitter bot that alerts you when Kanye is using ALLCAPS on his blog.

"SUBJECT'S ALLCAPS BLOG POSTS ALERTED TO TWITTER"
Note: This bot monitored Kanye's old blog at kanyeuniversecity.com which no longer exists. The @kanyerants Twitter account remains as an archive of detected rants from 2009-2010.

How it worked

The Ruby script would:

  • Scrape Kanye's blog for new posts
  • Check if the text was ALL CAPS (the kanye?(text) function)
  • If a rant was detected, shorten the URL via bit.ly and post to Twitter
  • Store found rants in a SQLite database to avoid duplicates
